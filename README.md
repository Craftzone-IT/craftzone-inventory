# craftzone-inventory
# Raktárkészlet kezelő

Önállóan hosztolható, nyílt forráskódú raktárkészlet-kezelő webalkalmazás.

## Funkciók

- 📦 Termékek és kategóriák kezelése
- 📊 Készletmozgások követése
- 📈 Riportok és statisztikák
- 👥 Többfelhasználós hozzáférés (jogosultságkezeléssel)
- 🇭🇺 Magyar nyelvű felület

## Technológiák

- PHP 8+
- MySQL / MariaDB
- Vanilla JavaScript
- HTML5 / CSS3
